# store5

## 开发

```bash
# 安装依赖
$ npm i
$ npx lerna bootstrap

# 启动自动构建服务
$ npm run watch
# 启动web服务
$ npm run server
```

## 修正代码格式、检查静态语法

```bash
$ npm run prettier
$ npm run eslint
```

## 构建

```bash
$ npm run build
```

## 查看更改

```bash
$ npx lerna changed
```

## 发布

```bash
$ npx lerna publich
```
